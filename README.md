Travel Planner AI Agent



This project is a conversational AI agent that creates complete travel plans from a simple user request. You can enter something like “Plan a 3-day trip to Kochi with a budget of 30000,” and the agent will understand the requirements, ask for any missing details, and generate a structured itinerary with flights, hotels, places to visit, and weather.



It is built using LangGraph and Groq-hosted LLaMA models, with MCP-based tools for data retrieval. The agent follows a step-by-step reasoning loop to decide what information to fetch and combines everything into a clear, final travel plan.
